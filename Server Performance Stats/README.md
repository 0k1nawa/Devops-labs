# Server Perfomance Stats
A bash script to analyse Server perfomance stats such as:

***• CPU usage***<br>
***• Memory usage***<br>
***• Disk usage***<br>
***• 5 Top processes by CPU consumption***<br>
***• 5 Top processes by Memory consumption***<br>

## Getting started
 **1) clone the repository**
```
git clone https://github.com/0k1nawa/Devops-labs.git
```

 **2) grant execution permission**
 ```
chmod +x Devops-labs/'Server Perfomance Stats'/ServerStats.sh
```
 **3) run the script**
 ```
./Devops-labs/'Server Perfomance Stats'/ServerStats.sh
```



This project is part of [roadmap.sh](https://roadmap.sh/projects/server-stats) DevOps projects.
